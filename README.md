SQL Injection Security Assessment
Overview

This project demonstrates the identification, analysis, and exploitation of SQL Injection (SQLi) vulnerabilities within a deliberately vulnerable web application deployed in a controlled laboratory environment.

The assessment combines theoretical research with practical penetration testing to demonstrate how SQL Injection can lead to authentication bypass, database enumeration, and sensitive data exposure.

Note: This project was completed solely for educational purposes. All testing was performed in an isolated virtual lab using synthetic data.
